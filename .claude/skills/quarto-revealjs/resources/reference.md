## Quick Reference

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `R` | Toggle scroll view |
| `S` | Speaker notes |
| `M` | Menu |
| `?` | Help |

### URL Parameters

- `?view=scroll` - Scroll mode
- `?view=print` - Print view
